# NOTES

## FOR CONSIDERATION AND ADDITION to CLAUDE.MD WHERE APPROPRIATE

1. Make changes
2. Typecheck (fast)

- bun run typecheck

3. Tests should be written wherever possible
4. Run tests
5. Lint before committing
6. Markdown files should follow the Google Markdown Style Guide
7. Use `bun` for running scripts and managing dependencies. Avoid `npm` and `yarn`.
8. Allowed tools

- Bash(bq query:\*)
- Bash(bun run build:\*)
- Bash(bun run lint:file:\*)
- Bash(bun run test:\*)
- Bash(bun run test:file:\*)
- Bash(bun run typecheck:\*)
- Bash(bun test:\*)
- Bash(cc:\*)
- Bash(comm:\*)
- Bash(find:\*)
